SmartFoxServer 2X examples

Before proceeding...

Please notice that all examples are configured to connect locally (127.0.0.1): this means that you can only run them from the same computer on which SFS2X is installed. If you try to use them from another machine, they won't connect. You can however make the Flash examples work by simply changing the IP address contained in the sfs-config.xml file available in each example folder. For the Unity examples you will need to get the source code, recompile and re-deploy them after changing the setting in the code.
